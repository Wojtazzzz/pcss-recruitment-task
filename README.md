## Installation

1. Clone the repository
```sh
gh repo clone https://github.com/Wojtazzzz/pcss-recruitment-task.git && cd pcss-recruitment-task
```

2. Install app
```sh
npm install 
```

3. Run dev server (http://localhost:5173/)
```sh
npm run dev
```

4. Or build app
```sh
npm run build
```
